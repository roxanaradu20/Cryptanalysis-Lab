This project implements the Pohlig–Hellman algorithm to efficiently compute discrete logarithms modulo a prime. It combines prime factorization, the Baby-Step Giant-Step algorithm, and the Chinese Remainder Theorem to solve 𝛼 ≡ β(mod p). The code is especially efficient when p−1 is smooth (factors into small primes). It demonstrates computing the discrete logarithm of a number in a multiplicative group modulo a prime, which is a key concept in cryptanalysis of Diffie–Hellman and related cryptosystems.
