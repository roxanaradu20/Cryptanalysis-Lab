This project implements the Vigenère cipher encryption and demonstrates its cryptanalysis. The program first preprocesses a text file by removing non-letter characters and converting all letters to uppercase. It can encrypt the plaintext using a random or predefined key. For cryptanalysis, the program estimates the key length using the Index of Coincidence (IC) and then recovers each key letter by calculating the Mutual Index of Coincidence (MIC) with standard English letter frequencies. Multiple runs with different keys illustrate the effectiveness of this statistical attack, showing how classical frequency analysis can compromise the Vigenère cipher.
